### 列 5：运镜+主画面描述(含台词) — 核心列

这是表格中最重要的列。所有描述必须是**可以被摄影机直接拍摄的视觉事实**。

#### 5.1 三要素强制格式

**必须强制以中括号 `[...]` 开头，括号内必须包含以下三要素，严禁漏写：**

```
[摄影机视角/高度, 构图景别/特殊视角, 摄影机运动方式]
```

**格式示例**：
- `[微俯视, 全景, 摇臂下降]` — 完整三要素
- `[平视, 中特写, 固定镜头]` — 固定机位
- `[微仰拍, 七分身镜头, 缓缓推轨]` — 推轨运动
- `[低角度仰拍, 大远景, 缓慢推进]` — 低角度+推进
- `[平视, 过肩镜头, 固定镜头]` — OTS + 固定
- `[正俯拍, 动作细节特写, 固定镜头]` — 极致俯拍
- `[俯拍 -> 平视, 全景 -> 特写, 伸缩摇臂下降]` — 复合运动，视角和景别在运动中变化

**若涉及过肩（OTS）、带关系单人（Dirty Single）等特殊视角，将其归入第二要素（景别/特殊视角）中。**

#### 5.2 场景首镜站位（强制规则）

**每个场景的第一个镜头**，必须在画面描述开头，用一句话明确交代所有出场角色的初始站位和朝向：

> `【场景首镜站位】: A位于[位置]，面向[方向]；B站在[位置]，面向[方向]，两人相距约[X]米。`

后续镜头**不再重复站位**，仅在位置发生变化时描述位移（如 `【站位位移】: 林晓彤走到梳妆台旁。`）。

#### 5.3 场景空间逻辑与摄影机轴线

每场戏拆镜前，必须先静默建立空间逻辑，再选择机位。空间逻辑至少包括：

- 主要表演区域、入口/出口、障碍物、关键道具位置和可见光源。
- 角色初始站位、朝向、彼此距离、视线方向和可移动路线。
- 摄影机轴线：通常以主要对话对象之间的连线、人物行进方向或危险来源方向作为轴线。

**轴线规则**：
- 正反打、过肩、带关系单人和双人同框必须保持同一侧轴线，确保人物视线方向和左右关系连续。
- 如果剧情需要越轴，必须用一个空间重建镜头解释：角色跨过轴线、摄影机跟随移动、全景重新建立方位，或危险/道具改变了新的观看轴线。
- 场景首镜站位必须来自该空间逻辑，后续位移必须能从上一镜结束状态连续到下一镜开场状态。
- 行动路线不能凭空跳转。角色走到、靠近、后退、进入、离开、绕到另一侧时，运镜列必须写 `【站位位移】` 或在主画面中清楚展示移动过程。

#### 5.4 对话场景镜头策略

**双人对话**：
- 优先使用正反打系统：A 的近景/中近景与 B 的近景/中近景在同一轴线两侧形成清楚的视线关系。
- 过肩镜头用于确认对话空间；带关系单人用于保留对方身体边缘和压力；纯净单人用于强调孤立或决断。
- 情节紧凑、台词很短或情绪需要同框压迫时，可以使用双人同框。双人同框必须写清两人距离、朝向和画面关系，例如：`两人相距约半米，林晓彤面向沈夜，沈夜侧身避开她的视线。`
- 快速甩镜头只用于突然插话、视线迅速转移、危险被发现或短台词连续交锋；不得用甩镜头替代常规正反打。

**多人对话**：
- 先用全景或三人/群像镜头建立空间关系，再给关键发言者和关键反应者单独近景或中近景。
- 群像镜头只承担空间、压力、队形和同时反应，不替代关键角色的发言权或情绪变化。
- 多人同框中，除主体外的可见角色也必须有轻量状态：视线转向、退半步、手指收紧、沉住呼吸等。
- 对话密集时，优先按“发言者近景 → 被点名者反应 → 空间关系确认”组织，不把所有台词塞进一个平视中景。

#### 5.5 切镜密度指引

目标是中高密度剧情片拆镜：常规镜头 3-6 秒，短插入镜 1-3 秒。不要为了避免机械拆碎，把多个信息节点塞进一个镜头。

- **对话**：按发言权和反应切。新角色接话、问答关系变化、被点名者出现明显反应时，优先新镜；多人对话不要用一个群像镜头吃完整段台词。
- **动作**：按发起、接触、结果切。攻击、拦截、道具操作、开门、跳入、转身逃离等动作链，至少把“发起”和“结果”拆清。
- **道具与状态**：道具激活、变形、熄灭、显示数字、裂痕修复、符文亮起或暗下，必须用单独镜头或清楚的切点确认。
- **超自然变化**：按触发、变化、后果切。现实/闪回/灵魂层切换、阵法变化、祭池变化、锁魂柱状态变化，不得与前后对话或普通反应混在一个长镜头内。
- **空间方向**：角色分头行动、视线转到新危险源、摄影机轴线改变、从中央切到东/西/南侧时，先用新镜重建空间关系。

#### 5.6 画面描述撰写逻辑链

按以下逻辑链撰写，所有描述必须是可以被摄影机直接拍摄的视觉事实：

```
[摄影机视角与高度] → [构图取景与景别] → [摄影机运动方式] → [主体动作与表演] → [嵌入台词]
```

**视觉描写戒律（最高优先级）**：
- **只写看得见的**：只描述形状、颜色、光线、阴影、速度、方向、身体动作、面部变化。
- 摄影机无法拍摄"内心""气氛""命运""感觉"等抽象概念——必须将其翻译为具体可见的视觉画面。
- 情绪表达用可见的肢体反应：瞳孔收缩、眉头紧皱、身体紧绷、身体颤抖、微微挑眉、深呼吸、下颌咬紧、猛然转身、缓缓低头、缓缓抬头、手指收紧、指关节发白等。
- 禁止使用比喻词："仿佛""犹如""宛如""好似"。
- 禁止使用营销辞藻："令人叹为观止""视觉盛宴""光影交响""完美呈现"等。

**台词嵌入格式**：
- 将台词融入动作尾端：`动作描写，"台词内容。"`
- 多角色同场时明确说话对象：`A转向B，动作描写："台词内容。"`
- 禁止使用脱离表演动作的纯说明句式（如 `A说: ...` `B冲着A喊道: ...`）。

#### 5.7 连续性铁律

- 每个镜头的开场状态必须**严格继承上一镜头的结束状态**。
- 状态不允许跳跃或重置：角色的位置、朝向、情绪状态、道具位置必须连续。
- 发生位移时，必须在镜头中展示移动过程。
- 切换镜头后，仍需重新交代必要的角色位置和朝向。
- 前后镜头不得互相打架：同一角色的左右方位、视线方向、互动对象和移动方向必须能连续解释。

## 摄影机语言参考 (四类体系)

以下术语体系必须在撰写「运镜+主画面描述」列时内化使用。选择最契合大卫·芬奇/维伦纽瓦风格的运镜：克制、精准、每一帧都有信息增量。

### 一、摄影机视角与高度 (Camera Angles & Height)

| 术语 | 英文 | 用法说明 |
|------|------|----------|
| 平视镜头 | Eye-Level Shot | 与角色视线持平，中性客观，最常用 |
| 俯拍 | High Angle Shot | 机位高于主体，表现弱小/脆弱/被压制 |
| 微俯视 | Slightly High Angle | 略高于视线，轻度审视感 |
| 仰拍 | Low Angle Shot | 机位低于主体，表现强大/威胁/英雄感 |
| 微仰拍 | Slightly Low Angle | 略低于视线，轻度仰视感 |
| 正俯拍 / 鸟瞰 | Overhead / Bird's Eye View / Top-Down | 90°垂直向下，表现孤立/宿命/全知视角 |
| 极低仰拍 | Worm's Eye View | 贴地向上，极端夸张的压迫或宏伟感 |
| 荷兰角 | Dutch Angle / Dutch Tilt / Oblique Angle | 画面倾斜，表现失衡/压迫/混乱/危险/眩晕 |
| 贴地镜头 | Ground-Level Shot | 机位紧贴地面 |
| 膝盖高度镜头 | Knee-Level Shot | 机位在膝盖高度 |

### 二、构图取景与景别 (Shot Sizes / Framing)

| 术语 | 英文 | 用法说明 |
|------|------|----------|
| 大远景 | Extreme Wide Shot (EWS) / Extreme Long Shot (ELS) | 人物极小，环境主导，建立空间感/孤立感 |
| 全景 | Wide Shot (WS) / Long Shot (LS) | 人物全身+环境，交代空间关系 |
| 全身镜头 | Full Shot (FS) | 人物从头到脚充满画面 |
| 七分身镜头 | Cowboy Shot / American Shot | 大腿中部到头顶 |
| 中景镜头 | Medium Shot (MS) | 腰部以上，常用对话镜头 |
| 中特写 | Medium Close-Up (MCU) | 胸部/肩部以上，常用反应镜头 |
| 特写镜头 | Close-Up (CU) | 面部充满画面 |
| 大特写 | Extreme Close-Up (ECU) | 仅眼睛/嘴唇/手部等局部 |
| 微距特写 | Macro Shot | 极致细节放大 |
| 插入镜头 | Insert Shot | 动作细节特写（如手部开门、看表、道具操作）|

### 三、人物关系与特殊视角 (Character Relations & Special POVs)

| 术语 | 英文 | 用法说明 |
|------|------|----------|
| 主观视角 | Point of View (POV) | 从角色眼睛出发，观众看到角色所看 |
| 过肩镜头 | Over the Shoulder (OTS) | 越过A的肩膀拍摄B，建立对话空间关系 |
| 带关系单人 | Dirty Single | 前景带有一点对面人物的边缘，拍单人 |
| 纯净单人 | Clean Single | 画面中完全没有对面人物 |
| 正反打 | Shot / Reverse Shot | 双人对话的优先基础结构，保持同一轴线和视线方向 |
| 双人镜头 | Two Shot | 两人同框 |
| 双人同框压迫 | Tight Two Shot | 两人距离很近或被空间挤压，必须写清距离、朝向和前后/左右关系 |
| 三人镜头 | Three Shot | 三人同框 |
| 群像镜头 | Crowd Shot | 多人同框 |
| 单人近景 | Single Close Shot | 多人对话中确认关键发言者或关键反应者的独立情绪 |
| 纯侧脸镜头 | Profile Shot | 主体完全侧面 |
| 视线留白构图 | Looking Room / Lead Room | 主体视线方向留出空间 |
| 头部留白构图 | Headroom | 控制头顶空间量 |
| 框中框构图 | Frame in Frame | 利用门窗/镜子等自然框景构图 |

### 四、摄影机运动方式 (Camera Movements)

#### 基础与光学运动 (Basic & Optical)

| 术语 | 英文 | 用法说明 |
|------|------|----------|
| 水平摇摄 | Pan Left / Pan Right | 机身不动，水平旋转 |
| 垂直俯仰摇摄 | Tilt Up / Tilt Down | 机身不动，垂直旋转 |
| 变焦推拉 | Zoom In / Zoom Out | 光学变焦改变视野 |
| 极速变焦 | Crash Zoom / Snap Zoom | 极快速变焦，制造突然感/惊吓 |

#### 物理位移运动 (Physical Movements)

| 术语 | 英文 | 用法说明 |
|------|------|----------|
| 移动推轨 / 移动拉轨 | Dolly In / Dolly Out | 摄影机沿轨道物理靠近/远离主体 |
| 横移跟拍 | Track Left / Track Right (Trucking) | 水平方向横向移动 |
| 垂直升降 | Pedestal Up / Pedestal Down (Ped) | 摄影机垂直升降 |
| 环绕运镜 | Arc Shot | 围绕主体 180° 或 360° 旋转 |
| 跟随镜头 | Following Shot / Tracking Shot | 跟随主体移动的所有运镜统称 |
| 缓慢推进 / 缓慢拉出 | Push In / Pull Out | 语感上比 Dolly 更强调主观情感的靠近或疏离 |

#### 特殊设备与复杂运动 (Specialized & Complex Movements)

| 术语 | 英文 | 用法说明 |
|------|------|----------|
| 手持拍摄晃动 | Handheld Shooting / Handheld Shot | 手持微晃，增强临场感/混乱/紧张 |
| 斯坦尼康平稳跟随 | Steadicam Shot | 平稳流畅的移动跟随 |
| 摇臂镜头 | Crane Shot / Jib Shot | 摇臂升降运动 |
| 伸缩摇臂运镜 | Technocrane Movement / Telescopic Crane | 大范围伸缩摇臂 |
| 航拍 / 无人机 | Aerial Photography / Drone Shot | 空中拍摄 |
| 希区柯克变焦 | Dolly Zoom / Vertigo Effect / Zolly | 推轨+反向变焦，创造空间扭曲的眩晕感 |
| 甩镜头 | Whip Pan / Swish Pan | 极快速水平摇摄产生运动模糊，用于突然切换/发现 |
| Z轴翻转镜头 | Roll / Z-Axis Rotation | 画面绕光轴旋转 |
| 机械臂轨迹运动 | Motion Control (MoCo) | 精确重复轨迹的机械臂运动 |
| 索道摄影 / 飞猫运镜 | Cable Cam Shot / Wire Cam | 索道高速滑行拍摄 |

#### 运动格式示例

- **运动转换**：`[俯拍 -> 平视, 全景 -> 特写, 伸缩摇臂下降] 摄影机从高空45°俯拍，匀速下降到桌面高度。`
- **固定状态**：`[平视, 特写, 固定镜头], [过肩镜头]。摄影机越过A的肩膀对准B，B说完台词，焦点快速切换到A的肩膀。`
- **复合运动**：`[平视, 双人镜头, 横移跟拍] 摄影机横向移动，捕捉两人的相对运动。`
- **手持**：`[平视, 纯净单人镜头, 手持拍摄晃动] 摄影机正对A的面部。手持微晃。A的下巴剧烈抽搐……`
- **正反打**：`[平视, 过肩中近景, 固定镜头] 摄影机越过林晓彤右肩拍沈夜，保持两人连线左侧轴线，沈夜视线向画面左侧回看她。`
- **双人同框**：`[平视, 双人中近景, 固定镜头] 摄影机在桌边侧前方，两人相距约半米，林晓彤正面压近，沈夜侧身避开视线。`
- **甩镜头**：`[平视, 双人近景 -> 单人近景, 快速甩镜头] 顾成短促插话，镜头从沈夜甩到顾成脸上，停在顾成收紧的下颌。`
