# Seedance Prompt Rules

本文件只用于导演主表锁定并通过自检后，派生 `Prompt` 列。当前版本采用稳定精简五字段，不再输出复杂 Seedance 长提示词，不生成关键帧或生图提示词。

## 适用边界

- Prompt 是后置派生列，不能修改镜号、Beat、场景、原文、时长、运镜主画面、备注或连续性。
- 如果 Prompt 派生时发现主表矛盾，必须返回主表自检阶段，不得在 Prompt 层静默修表。
- Prompt 只能使用当前镜头可见内容、已登记画外声、`visible_characters` 和 `visible_props`。

## 固定五字段

每个镜头的 Prompt 必须且只能包含五行：

```text
时间：0秒-X秒
景别：基础景别或特殊视角
构图：可见主体、同框角色和道具的位置关系
运镜手法：真实摄影运动
画面内容：可见动作、表情、道具互动、画内/画外声和原样对白
```

字段说明：

- `时间`：从 0 秒开始，到本镜 `duration_seconds` 结束。
- `景别`：来自运镜三元组第二项。
- `构图`：只写画面内可见主体和关系，使用一句短句。
- `运镜手法`：来自运镜三元组第三项。
- `画面内容`：忠实提取主画面中可拍动作、表情、道具、声源和对白。

## 禁止内容

Prompt 不得包含：

- `【镜内变化】`
- `【人物关系】`
- `【人物关系补充】`
- `【站位位移】`
- `【机位逻辑】`
- `【场景首镜站位】`
- `运镜修正`
- `关键帧`
- `生图`
- `AI生图提示词`
- `风格：`
- `禁止：`
- 审稿意见、制作说明、校验备注、过程痕迹

## 自检

输出前逐镜确认：

1. 五字段完整且只有五行。
2. Prompt 与已锁定主表一致。
3. 没有内部标签或关键帧内容。
4. 没有把画外人物写成可见表演。
5. 没有为了压缩时长而省略台词或动作。
