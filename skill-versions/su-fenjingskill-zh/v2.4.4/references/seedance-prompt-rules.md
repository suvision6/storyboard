# Seedance Prompt Rules

<!-- for skill-version: 2.4.4 / rule-revision: 2.4.4-emotion-performance-guard-2026-07-16 -->

仅在 **Gate B 最新轮次批准且 `reviewed_hash` 匹配、导演主表与审计字段冻结之后**阅读本文件，用于派生 `Prompt` 列。当前版本采用稳定精简五字段，不输出复杂长提示词，不生成关键帧或生图提示词。

## 适用边界

- Prompt 是后置派生列。**禁止修改 `SKILL.md` 第 0 节"受保护审计字段统一清单"中的任何字段**，也禁止修改镜号、场景、原文、时长、运镜主画面、备注。
- Prompt 派生时发现主表矛盾，必须**申请回流至 Gate B**（说明理由，经人工同意后修改并重新过门禁），不得在 Prompt 层静默修表。
- Prompt 只能使用当前镜头可见内容、已登记画外声、`visible_characters` 和 `visible_props`。

## 固定五字段

每个镜头的 Prompt 必须且只能包含五行：

```text
时间：0秒-X秒
景别：基础景别或特殊视角
构图：可见主体、同框角色和道具的位置关系
运镜手法：真实摄影运动
画面内容：可见动作、表情、道具互动、画内/画外声和原样对白
```

字段说明：

- `时间`：从 0 秒开始，到本镜 `duration_seconds` 结束。
- `景别`：来自运镜三元组第二项。**跨景别镜头（三元组含 `->` 且备注有 `[景别跨度]`）写 `起始景别→结束景别`**，例如 `全景→特写`；非跨景别镜头禁止使用箭头形式。
- `构图`：只写画面内可见主体、同框关系、位置关系和道具关系，使用一句短句。
- `运镜手法`：来自运镜三元组第三项。
- `画面内容`：忠实提取主画面中可拍动作、表情、道具、声源和对白。
- `画面内容` 必须保留锁定主画面中已批准的动作、表情、呼吸、声音质感和合法环境反馈，不得在 Prompt 层二次补写或删减。

字段职责铁律：

- `构图` 字段不得包含对白、画外声、声音来源、动作链或情绪反应；这些内容只能进入 `画面内容`。
- Prompt 中引号不闭合或字段职责串位视为 FAIL。
- **对白保真校验只比对引号内内容**；嵌入句式与引号标点不计入"改字"，但引号内的对白必须与主表和原文逐字一致。

## 长度控制

- 单镜 Prompt 五行总长超过 **800 字符**时，校验器触发 `prompt_overlength` WARN。
- 处置路径唯一：申请回流至 Gate B，由人工裁决拆镜或保留；人工裁决保留时，该 WARN 在 `warn_resolutions` 以 `resolved_by: "human"` 记录。
- **禁止以截断、省略或改写台词的方式规避长度告警**；也禁止为压缩时长省略动作。
- 不得套用六段式总提示词或 1900 字总上限；不得增加视听限制、角色设定、导演风格、美学流派、自动增强冲突或全局负面指令。

## 禁止内容

Prompt 不得包含：

- `【镜内变化】`
- `【人物关系】`
- `【人物关系补充】`
- `【站位位移】`
- `【机位逻辑】`
- `【场景首镜站位】`
- `运镜修正`
- `关键帧`
- `生图`
- `AI生图提示词`
- `风格：`
- `禁止：`
- 备注封闭清单中的任何标注（如 `[必拆相邻]`、`[反转动机]`、`[不可拆说明]` 等）
- 审稿意见、制作说明、校验备注、过程痕迹

## 自检

输出前逐镜确认：

1. 五字段完整且只有五行。
2. Prompt 与已锁定主表一致；未触碰任何受保护审计字段。
3. 没有内部标签、备注标注或关键帧内容。
4. 没有把画外人物写成可见表演；画外人物只出现在含该角色名和"画外声"的同一行。
5. 没有为了压缩时长或长度告警而省略、截断台词或动作。
6. `构图` 没有混入对白、声音或动作链；所有引号闭合；引号内对白与主表逐字一致。
7. 跨景别镜头的 `景别` 字段使用 `起始景别→结束景别` 且备注确有 `[景别跨度]`；两者缺一即 FAIL。
8. 超长 Prompt 已走 Gate B 回流裁决，未自行处理。
